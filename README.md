# Blob_detection
Detection of Blobs in WISPR Images Using a Deep Learning Algorithm.

To evaluate blobs in your set of data, you can follow these steps:

(1) First, you need to convert into .png the .fits files to which you wanna apply the method. Use the provided script to convert into png the fits files to ensure that the format is correct and recognised by the algoritm.

(2) Save the .png files in a folder, you should run the evaluation script using the best.pt model to that folder of .png. So you need to modify the script to account for the path you saved the images on and also modify the path to where is the best model saved.

(3) Run the code.

(4) Automatically, the outputs will be saved in the folder you have chosen.
